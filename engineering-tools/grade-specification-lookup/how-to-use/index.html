How to use the Grade Specification Lookup

Use this guide to select a governing steel grade, review its requirements, check test results, compare grades, run engineering calculations, and produce a traceable report.

Quick start

  1. Select the specification body, grade, product form, and PSL or category where shown.
  2. Enter the governing wall thickness. The tool resolves the correct mechanical breakpoint automatically.
  3. Choose SI or Imperial display units.
  4. Review chemistry, mechanical, Charpy/DWTT, product-form, testing, and notes cards.
  5. Activate Sour Service, Offshore, or Customer Supplement overlays only when they apply.
  6. Open Compliance Check, enter results manually or paste a batch, and run the check.
Reference only. Always verify the current controlled specification edition, purchase order, customer requirements, product form, manufacturing route, and supplementary requirements.

Understand the requirement cards

IndicatorMeaning
MIN ≥The entered result must be equal to or greater than the stated value.
MAX ≤The entered result must be equal to or less than the stated value.
Clause tagShows the specification table, clause, note, or equation supporting the value.
⚠ verifyThe value is structured but has not yet completed the controlled human audit.
—No numerical limit is specified in the selected record.
OverlayThe displayed value is changed by an active service or customer overlay.

Footnote superscripts link to chemistry rules such as combined alloy limits or Mn/C tradeoffs. Grade aliases such as X52, L360, 350W, and 50W can be found through global search.

Compliance checker

Manual entry

Enter the available chemistry, yield strength, tensile strength, elongation, specimen dimensions, individual CVN energies, test temperature, and hardness. Blank fields are not silently treated as passing results.

Verdicts

VerdictInterpretation
PASSAll checks pass using fully verified limits.
PASS WITH WARNINGSAll checks pass, but one or more applicable limits are still marked unverified.
FAILAt least one entered value or machine-readable footnote rule fails.
Current expanded records are intentionally unverified, so an acceptable result will normally display PASS WITH WARNINGS, not a clean PASS.

Failures are sorted first. Each result row shows the entered value, applicable bound, margin, clause reference, verification status, and overlay source where relevant.

Paste and batch mode

  1. Copy tabular data from Excel, Power BI, or a QADW query export.
  2. Paste tab- or comma-separated data into the batch area.
  3. Review detected columns and map each source column to an element or test field.
  4. Confirm the mapping, then run the batch.
  5. Review the row-level verdict table and open individual results as needed.

The most recent mapping is stored in the current browser so repeat exports from the same query format require less setup.

Service overlays

Available overlays are shown as toggle chips. They resolve at read time and never mutate the base grade record. Turning an overlay off returns the tool to the original specification values.

  • Sour Service: may tighten sulfur, hardness, chemistry, or testing requirements.
  • Offshore: may add toughness, testing, or supplementary requirements.
  • Customer Supplement: records customer-specific additions configured in the data.

The active overlay state is recorded in the printed compliance report.

Engineering calculators

Carbon equivalent

Enter chemistry to calculate CEIIW and Pcm. Both are displayed, and the selected grade's stored rule determines which value governs.

Sub-size Charpy

The tool follows the stored factor convention: full-size required energy × sub-size factor = sub-size required energy. A 10 × 5 mm specimen commonly uses the stored 0.50 factor where defined.

Elongation

For formula-based grades, enter strip width and thickness or round-bar diameter plus UTS. The tool computes area and the minimum elongation using the selected grade's stored parameters.

Hydrotest

Enter OD and wall thickness. The calculator uses resolved SMYS and the stored fiber-stress percentage to calculate pressure in SI and Imperial units.

Reverse lookup: What grade meets this?

Enter required minimum YS, optional UTS, optional CVN energy and temperature, and thickness. Results are sorted by the smallest positive YS margin, so the closest strength fit appears first.

Reverse lookup is a screening aid, not an equivalency determination. Chemistry, manufacturing route, delivery condition, product form, toughness, customer requirements, and edition applicability must still be reviewed.

Comparison, reports, export, and diagnostics

  • Comparison: pin the current grade, open a second grade, and review highlighted differences.
  • Favorites and recents: stored in local browser storage.
  • Print: use the specification datasheet or compliance report for a clean white-background job-file record.
  • Export: downloads the complete current SPEC_DATA object as JSON.
  • Import: replaces data for the current session only after validation passes.
  • Diagnostics: reports schema, breakpoint, footnote, overlay-path, enum, and unknown-field findings.

Troubleshooting

IssueAction
No grade shownClear global search and select the specification body again.
Form unavailableThe selected grade does not list that product form. Verify the governing specification and form.
No thickness rowCheck thickness units and confirm the value is inside the stored applicability range.
Import rejectedOpen Diagnostics and correct schema-version, required-field, path, or breakpoint errors.
Clean PASS never appearsThe applicable limits are still marked unverified; PASS WITH WARNINGS is expected.
Saved data missing elsewhereFavorites, recents, mappings, and draft inputs are browser-local and do not automatically sync.
