The investigation sequence

Each tool answers a different question. The output of one tool should determine the next move.

1. FlowchartWhere can the issue originate?
2. Check SheetHow will evidence be collected?
3. StratificationWhich subgroup carries the issue?
4. ParetoWhat deserves priority?
5. FishboneWhat could cause it?
6. HistogramWhat is the distribution doing?
7. Scatter PlotDo X and Y move together?
8. Control ChartIs the process stable and did the fix hold?
Terminology note: published lists differ on whether stratification, flowchart, or run chart occupies the seventh position. This lesson preserves all eight practical capabilities because both process mapping and stratification are essential in real investigations.
