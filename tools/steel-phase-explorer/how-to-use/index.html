How to Use the Steel Phase & Transformation Explorer

Before you begin

Purpose and limits

The explorer combines equilibrium phase analysis, transformation-path visualization, generalized kinetic diagrams, chemistry calculations, and training tools in one browser-based application.

Best uses

Training, concept development, phase-field identification, lever-rule practice, multi-point comparison, heat-treatment visualization, and preliminary trend screening.

Not a replacement for

Controlled standards, grade-specific TTT/CCT diagrams, validated thermodynamic software, laboratory testing, procedure qualification, or responsible engineering approval.

Engineering caution: Fe–Fe3C boundaries are simplified. TTT/CCT curves, kinetic fractions, property ranges, and weldability indicators are educational estimates and must not be used alone for material release, certification, or process qualification.

Fastest workflow

Quick start

Select temperature units

Choose Metric (°C) or Imperial (°F). Carbon remains in weight percent in both modes.

Enter a point

Type carbon and temperature, select a preset, click the diagram, or drag the active marker.

Read the result

Review the phase field, phase fractions, tie line, calculation, and schematic microstructure.

Add comparisons

Select + Add point. Every completed point stays plotted and is evaluated independently.

Open another module

Use the tabs for heat-treatment paths, TTT/CCT, chemistry and properties, or learning and export.

Suggested first exercise: plot 0.20 wt% C at 900 °C, 750 °C, and 700 °C to see the same steel move from austenite to the intercritical region and then below A1.

Module 1

Equilibrium diagram

Plot and manage points

  • Enter carbon from 0 to 1.2 wt% and a temperature inside the displayed range.
  • Select a point from its input row, diagram marker, or result card.
  • Click or drag on the diagram to move only the active point.
  • Use + Add point, Remove, and Reset to manage the point set.

Lever-rule results

Within a two-phase field, equilibrium mass fractions are calculated from the tie-line endpoints. For ferrite plus austenite:

fα = (Cγ − C₀) / (Cγ − Cα)    and    fγ = (C₀ − Cα) / (Cγ − Cα)

The fraction bar shows the result visually. A single-phase point displays 100% of the identified phase.

Microstructure panels

The live viewer is schematic, not photomicrographic. Toggle grain boundaries, carbon distribution, and labels. The final equilibrium panel estimates ferrite, pearlite, and proeutectoid cementite at room temperature.

Remember: equilibrium phases at a selected temperature are not automatically the same as final room-temperature microconstituents. Actual cooling path and kinetics matter.

Module 2

Heating and cooling paths

This module creates a temperature-versus-time cycle and animates the expected state as the cycle progresses.

Preset cycles

Choose normalizing, full annealing, process annealing, spheroidizing, austenitize and quench, quench and temper, intercritical annealing, austempering, or martempering. The step table and chart update automatically.

Custom cycle

  1. Enter carbon content.
  2. Select + Step.
  3. Set target temperature, duration, and mode for each segment.
  4. Use Play, Pause, or the time slider.

Read the output

  • The graph shows temperature history.
  • The timeline lists boundary crossings and expected events.
  • The live phase state reports the current equilibrium field.
  • The animated schematic follows the selected cycle position.

Crossing an equilibrium boundary does not prove instantaneous transformation. Time, alloying, austenite grain size, section thickness, and real cooling rate control the transformation response.

Module 3

TTT and CCT

TTT mode

Use the isothermal hold to see how a constant-temperature path intersects idealized start and finish curves.

CCT mode

Use cooling rate to see how a continuous path passes through ferrite, pearlite, bainite, and martensite regions.

Inputs

  • Cooling rate changes the continuous-cooling path.
  • Isothermal hold sets transformation time in TTT mode.
  • Final temperature affects the estimated amount transformed below Ms.
  • Chemistry from the chemistry module shifts the generalized curves and critical temperatures.

Outputs

Review estimated constituent fractions, critical temperatures, and the interpretation note. Use these results to compare trends—not to qualify a real grade.

The displayed curves are not certified grade-specific diagrams. Real data depend on exact chemistry, austenitizing practice, prior microstructure, grain size, test method, and section size.

Module 4

Chemistry and properties

Presets and custom chemistry

Select an illustrative preset for AISI 1018, 1045, 1080, 4130, 4140, or API X70 and choose Apply preset. For actual work, replace every value with verified heat chemistry.

Property estimates and comparison

Move the cooling-rate slider to see directional changes in hardness, strength, ductility, and toughness. The comparison table evaluates all plotted points using the active chemistry.

Data caution: verify that the carbon value plotted on the phase diagram and the carbon entered in the chemistry module refer to the same intended steel.

Module 5

Learn and export

Learning mode

Select New challenge, choose a phase field, and review the feedback. Use Show hint for guidance. The boundary guide explains A1, A3, Acm, and the ferrite solvus.

Save and share

  • Save scenario stores the current state in the same browser.
  • Load saved restores the last local scenario.
  • Copy share link embeds scenario state in the URL.

Export

  • CSV exports points, phase fields, fractions, and final microstructure estimates.
  • PNG exports the equilibrium diagram.
  • Print / PDF opens the browser print workflow.

For technical reporting, include chemistry, units, thermal assumptions, model limitations, data source, and analysis date with exported results.

Practical applications

Recommended workflows

Teach the diagram

Use one carbon level, add multiple temperatures, connect points, and discuss each boundary crossing.

Compare carbon levels

Hold temperature constant and compare phase fractions and final equilibrium microstructures.

Review heat treatment

Play a preset cycle, pause at critical events, and compare with the kinetics module.

Screen chemistry trends

Enter verified heat chemistry, review CE/Pcm and critical temperatures, then vary cooling rate.

Prepare training

Build a scenario, copy the share link, and ask learners to predict the result before opening it.

Create an appendix

Export CSV and PNG, document assumptions, and attach the controlled chemistry and process source.

Accuracy controls

Interpretation cautions

  • Equilibrium versus kinetics: Fe–Fe3C identifies equilibrium phases; TTT/CCT addresses time-dependent products.
  • Bainite and martensite: these are not equilibrium phase fields on the Fe–Fe3C diagram.
  • Phase versus microconstituent: ferrite and cementite are phases; pearlite and bainite are microconstituents.
  • Range: the equilibrium plot is limited to 0–1.2 wt% C and does not show the full cast-iron region.
  • Alloy effects: alloying shifts boundaries and kinetics; the plain-carbon equilibrium approximation is incomplete for alloy steels.
  • Properties: grain size, precipitation, inclusions, texture, prior deformation, section size, cooling uniformity, and test orientation matter.
  • Weldability: CE and Pcm are screening indicators. Heat input, hydrogen, restraint, thickness, preheat/interpass control, and procedure qualification remain essential.

Before relying on an output

Final checklist

  • Confirm carbon and alloy chemistry.
  • Confirm temperature units.
  • Separate equilibrium phases from kinetic products.
  • Document the thermal path.
  • Verify grade-specific TTT/CCT data.
  • Record section size and process history.
  • Treat properties as directional estimates.
  • Include model limitations in reports.
  • Verify with testing or validated software.
  • Obtain technical approval where required.
